Making Android App in Windows Shell

Setup

  • Download JDK1.6 means Java Development Kit Version 1.6.Install it.
  • Download JRE1.6 means Java Runtime Environment. Install it.
  • Next, select any directory where you will make your project.
  • In that directory, make 5 sub-directories:- "java","class","jar","dex","res".
  • Concept is that you will make java programs as source code and keep in 'java' directory.
  • Using 'javac' command you will compile into ".class" files and keep in "class" directory.
  • While compiling you may need external libraries in '.jar' files. So keep them in 'jar' directory.
  • Then you will use 'dx' command to combine all class files into .dex file and keep in 'dex' directory.
  • Lastly you will make an '.apk' file from '.dex' file & '.jar' external libraries.
  • Using 'adb' command, you transfer the apk file app to ur mobile using usb data cable
